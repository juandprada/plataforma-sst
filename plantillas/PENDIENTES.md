# Formatos pendientes / no reautorados

Formatos demasiado complejos para reautorar como plantilla HTML sencilla. Se
mantienen aparte para **edición manual** y no se ofrecen (aún) en la plataforma.

- **Plan de Emergencias** — extenso, con mapas/diagramas; edición manual.
- Matrices grandes (p.ej. IPEVR / matriz de riesgos) — muchas columnas y celdas
  combinadas; requieren maquetación dedicada.
- Formatos `.xlsx` tipo hoja de cálculo (cuestionario, evaluación inicial,
  presupuesto, indicadores) — se evaluarán como tablas HTML en una fase posterior.

> Al reautorar uno de estos, agrégalo a `manifest.json` y quítalo de esta lista.
